Resource Profile: DiagnosticReport: Laboratory Report

Official URL: http://hl7.eu/fhir/laboratory/StructureDefinition/DiagnosticReport-eu-lab Version: 0.2.0-ci
Standards status: Trial-use Maturity Level: 2 Computable Name: DiagnosticReportLabEu

Copyright/Legal: Used by permission of HL7 Europe, all rights reserved Creative Commons License

DiagnosticReport used to represent an entry of a Laboratory Report, including its context, for the scope of the HL7 Europe project.

Previous results

This version of the guide uses the supportingInfo extension to the Observation lab results, for conveying the previous.

Implementers are invited to provide their feedback about the possible adoption of the supportingInfo extension to this DiagnosticReport profile

Adding a PDF, images or other kind of additional data in a DiagnosticReport

In some cases it is requested to include PDF, images or other kind of additional data in a Diagnostic Report.

If the PDF to be added is the representation of the entire Laboratory report, then this will be recorded in the optional element presentedForm

In case of images or other kinds of additional data (that can be a pdf document) used as support for the interpretation of the report, this is realized by using the optional element media.

Usage:

  • Use this Resource Profile: Bundle: Laboratory Report
  • Refer to this Resource Profile: Composition: Laboratory Report
  • Examples for this Resource Profile: DiagnosticReport/dr-lab-example
