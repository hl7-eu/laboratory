Resource Profile: Patient: Animal

Official URL: http://hl7.eu/fhir/laboratory/StructureDefinition/Patient-animal-eu-lab Version: 0.2.0-ci
Standards status: Trial-use Maturity Level: 2 Computable Name: PatientAnimalEu

Copyright/Legal: Used by permission of HL7 Europe, all rights reserved Creative Commons License

This profile defines how to represent an Animal as subject of care in FHIR for the purpose of this guide. This is used to identify the species when a specimen is collected from an animal

Usage:

  • Use this Resource Profile: Bundle: Laboratory Report
  • Refer to this Resource Profile: Composition: Laboratory Report, DiagnosticReport: Laboratory Report, Observation Results: laboratory, ServiceRequest: Laboratory Order and Specimen: Laboratory
  • Examples for this Resource Profile: Patient/Patient-animal-example
